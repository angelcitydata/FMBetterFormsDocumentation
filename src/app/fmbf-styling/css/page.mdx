## What This Code Includes

### Global Styles & Resets
- **Body & Layout** - Sets up the main layout structure with proper overflow handling
- **Typography** - Configures DM Sans font family across all headings and body text
- **Heading Sizes** - Resets heading sizes (h1-h6) for consistent typography
- **Paragraph Spacing** - Adjusts paragraph margins for better readability

### Utility Classes

#### `.no-scrollbar`
Hides scrollbars across all browsers while preserving scroll functionality:
- Works in Firefox, Chrome, Safari, Edge, and Opera
- Maintains scroll behavior without visual scrollbars

#### `.visually-hidden`
Accessible way to hide content visually while keeping it available for screen readers:
- Replaces the old `.sr-only` class
- Follows modern accessibility best practices

#### `.outline-debug`
Development utility for visualizing element boundaries:
- Adds outlines to all child elements
- Helpful for debugging layout issues

### Component Styling

#### Multiselect Components
Complete styling for multiselect dropdown components:
- Custom colors for selected and highlighted options
- Disabled state styling
- Border and background customization
- Responsive font sizing

#### Navigation Customization
Sidebar and navigation menu styling:
- Active state colors using CSS variables
- Hover effects with smooth transitions
- Focus states for accessibility
- Icon color inheritance

#### Table Styling
VueTables and standard table formatting:
- Header background colors
- Row borders and spacing
- Column sorting icon positioning
- Text colors using CSS variables

### CSS Variables Used

The code references these CSS variables defined in the DOM Header:
- `--background-color` - Sidebar and general background
- `--white-color` - Primary white/light background
- `--primary-color` - Brand primary color for links and active states
- `--text-primary` - Primary text color
- `--radius` - Border radius for rounded corners

## Customization

You can customize various aspects of this CSS to match your needs:

### Change Multiselect Colors

```css
/* Modify highlight colors */
.multiselect__option.multiselect__option--highlight {
    background-color: #your-color; /* Change from green (#34D399) */
}

.multiselect__option.multiselect__option--highlight.multiselect__option--selected {
    background-color: #your-color; /* Change from red (#F87171) */
}
```

### Adjust Typography Sizes

```css
/* Modify heading sizes */
h1 { font-size: 2em; }      /* Make larger */
h2 { font-size: 1.75em; }   /* Make larger */
```

### Customize Navigation

```css
/* Change navigation hover effect */
.theme-default .sidebar-left .nav > li > a:hover {
  color: var(--your-custom-color) !important;
  background-color: var(--your-custom-bg) !important;
}
```

## Important Notes

### Font Family Dependency
This CSS uses the **DM Sans** font family, which is loaded in the DOM Header code. Make sure you've applied the [DOM Header code](/fmbf-styling/dom-header) first.

### CSS Variable References
Many styles reference CSS variables defined in the DOM Header's theme system. The CSS won't work correctly without the DOM Header code applied.

### Browser Compatibility
All styles are compatible with modern browsers:
- ✅ Chrome 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ Edge 90+

## Development Tools

### Debugging Layouts

Use the `.outline-debug` class during development to visualize element boundaries:

```html
<!-- Add to any container to debug child elements -->
<div class="outline-debug">
  <!-- All children will have visible outlines -->
</div>
```

### Hiding Scrollbars

Use `.no-scrollbar` when you want scrollable content without visible scrollbars:

```html
<!-- Scrollable content without scrollbars -->
<div class="overflow-auto no-scrollbar">
  <!-- Your scrollable content -->
</div>
```

### Accessible Hidden Content

Use `.visually-hidden` for screen-reader-only content:

```html
<!-- Visible to screen readers, hidden visually -->
<span class="visually-hidden">Additional context for screen readers</span>
```
