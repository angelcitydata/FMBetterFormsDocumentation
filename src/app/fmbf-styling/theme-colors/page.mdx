## Usage Guidelines

### CSS Variable Access

All theme colors are available as CSS custom properties:

```css
/* Use theme colors in your custom CSS */
.my-element {
  background-color: hsl(var(--background));
  color: hsl(var(--foreground));
  border-color: hsl(var(--border));
}

/* Status colors */
.success-message {
  background-color: hsl(var(--success));
  color: white;
}
```

### Tailwind Classes

Theme colors are integrated with Tailwind CSS:

```html
<!-- Background and text colors -->
<div class="bg-card text-card-foreground">Card content</div>

<!-- Interactive elements -->
<button class="bg-primary text-primary-foreground hover:bg-primary-hover">
  Primary Button
</button>

<!-- Borders and focus states -->
<input class="border-input focus:ring-ring" />
```

### Dark Mode

All theme colors automatically adapt to dark mode when using the `data-theme="dark"` attribute or `dark` class on the root element. The color values are carefully calibrated for optimal contrast and accessibility in both light and dark themes.
